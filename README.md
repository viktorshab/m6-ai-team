# AI-команда для бизнеса

Готовая команда AI-ассистентов, которые работают как сотрудники: оркестратор **Универсал** раскладывает твою задачу на шаги и раздаёт их помощникам, а каждый помощник делает свой кусок и оставляет файл-результат.

Это не один большой промпт, а набор ролей, которые ты ставишь себе один раз и зовёшь по имени в любом проекте.

## Установка (один раз)

Открой свою папку проекта в **Claude Code** или **Codex** и напиши нейронке:

```
Установи мне глобально команду AI-ассистентов из этого репозитория:
https://github.com/viktorshab/m6-ai-team
Это оркестратор «Универсал» и его помощники. Поставь все скиллы глобально,
чтобы я мог звать их по имени в любом проекте.
```

Нейронка скачает и поставит скиллы сама. Если хочешь поставить вручную одной командой:

```bash
npx skills add viktorshab/m6-ai-team --all -g --agent claude
```

Скиллы лягут в твою глобальную папку (`~/.claude/skills/`) и будут доступны во всех проектах.

## Как пользоваться

Открываешь папку проекта и зовёшь нужного по имени — объяснять, кто он, не нужно:

```
Вызови Универсала. Нужно проработать: хочу больше заявок из Instagram.
Что у меня есть: статистика постов и переписки с клиентами.
Сначала задай уточняющие вопросы, потом разложи задачу в маршрут и приступай.
```

Универсал разложит задачу на роли и подскажет, кого звать дальше. Любого помощника можно вызвать и напрямую: «Вызови Упаковщика…», «Вызови Аналитика…».

## Кто в команде

| Ассистент | Что делает | Как позвать |
|---|---|---|
| **Универсал** | раскладывает размытую задачу в маршрут, ведёт команду, собирает сводку | «вызови Универсала» |
| **Поиск информации** | достаёт конкретные боли и язык клиентов, изучает тему/инструмент | «вызови исследователя» |
| **Упаковщик воронок** | из болей собирает оффер и воронку | «вызови упаковщика» |
| **Проработка лендинга** | собирает логику посадочной страницы под цель | «вызови лендинг» |
| **Сборка MVP** | оценивает реальность идеи и собирает минимум для проверки | «вызови MVP» |
| **Аналитик** | из данных (реклама, звонки, CRM, продажи) достаёт решения | «вызови аналитика» |
| **Ищейка** | проверяет, какие инструменты и модели актуальны под задачу | «вызови Ищейку» |

## Безопасность — кто за что отвечает

Эти ассистенты готовят работу, но не действуют за тебя в опасных местах.

- **AI делает сам:** читает данные, анализирует, пишет черновики, собирает файлы-результаты.
- **Только ты (человек):** деньги и бюджеты, публикации наружу, сообщения клиентам, цены и гарантии, реальные доступы и ключи. AI готовит — решаешь и нажимаешь ты.

Свои ключи и доступы в скиллы вписывать не нужно — они работают с данными, которые ты даёшь сам.
